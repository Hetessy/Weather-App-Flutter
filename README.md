- Dark and Light Mode friendly
- Weather from API
- Icons based on weather from API
- Location is based on client IP
- Scalable
- Performance friendly
- Today weather info
- Forecast for today
- 3-day forecast
- Dynamic temperature color change based on temperature
- Null safety
